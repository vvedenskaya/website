# Project media

Put project preview images and/or videos here and reference them in `index.html`.

**Images** (screenshots): e.g. `nyc-accessible-places-scraper.png` — used as poster/fallback.  
**Videos**: e.g. `nyc-accessible-places-scraper.mp4` (or `.webm`) — optional; if present, use a `<video>` tag in the project window. Use `poster="images/...png"` so a frame shows before the video loads.

Recommended: about 320×240 px or 4:3 aspect ratio. For video: short loop, muted (autoplay works only when muted).
